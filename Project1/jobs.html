Position 1: Cloud Engineer

Reference No: CLD01

Reports To: Lead Cloud Architect

Salary Range: $90,000 – $115,000 per annum

Position Description

The Cloud Engineer designs, deploys, and manages scalable cloud infrastructure using AWS and Azure. The role involves automation, monitoring, and optimization of cloud systems to ensure high performance, cost efficiency, and strong security compliance.

Key Responsibilities

  • Build and maintain virtual networks, storage, and compute instances across multiple cloud providers.
  • Automate deployment and monitoring using tools like Terraform, Ansible, and Jenkins.
  • Collaborate with developers to optimize performance in cloud environments.
  • Ensure compliance with security and data governance standards.

Required Qualifications & Skills

Essential

  1. Bachelor’s degree in IT, Computer Science, or related discipline.
  2. 2+ years’ experience with AWS, Azure, or Google Cloud Platform.
  3. Strong knowledge of networking, virtualization, and Linux/Windows servers.
  4. Experience with Infrastructure as Code (IaC) and CI/CD pipelines.

Preferable

  • Certified AWS Solutions Architect or Azure Administrator Associate.
  • Experience with containerization (Docker, Kubernetes).
  • Proficiency in scripting languages (Python, Bash).

Position 2: Cybersecurity Analyst

Reference No: SEC02

Reports To: Security Operations Manager

Salary Range: $88,000 – $121,000 per year

Position Description

The Cybersecurity Analyst monitors, analyzes, and responds to potential threats across organizational systems. Responsibilities include using SIEM tools, investigating incidents, conducting vulnerability assessments, and improving defense mechanisms to protect company assets.

Key Responsibilities

  • Monitor system logs for suspicious activity using SIEM platforms.
  • Respond to and document security incidents and breaches.
  • Conduct vulnerability scans and risk assessments regularly.
  • Collaborate with IT teams to implement security improvements.

Required Qualifications & Skills

Essential

  1. Bachelor’s degree in Cybersecurity, IT, or Computer Science.
  2. Strong knowledge of network security, firewalls, and IDS/IPS systems.
  3. Experience with Splunk, Wireshark, or Nessus tools.
  4. 2+ years’ experience with AWS, Azure, or Google Cloud Platform.

Preferable

  • Certifications: CompTIA Security+, CEH, or CISSP.
  • Experience with cloud security platforms and compliance frameworks.
  • Understanding of ISO 27001 and NIST standards.

Position 3: Front-End Developer

Reference No: FED03

Reports To: UI/UX Lead Designer

Salary Range: $80,000 – $105,000 per year

Position Description

The Front-End Developer will design, build, and maintain responsive web interfaces that deliver outstanding user experiences. Working closely with designers and back-end engineers, this role requires strong knowledge of modern front-end technologies, accessibility standards, and performance optimization.

Key Responsibilities

  • Develop and maintain front-end features using HTML5, CSS3, and JavaScript (ES6+).
  • Implement responsive layouts and animations using frameworks such as React or Vue.js.
  • Collaborate with UX/UI designers to translate design mockups into functional interfaces.
  • Ensure accessibility, browser compatibility, and high performance across all devices.

Required Qualifications & Skills

Essential

  1. Bachelor’s degree in Computer Science, Software Engineering, or a related field.
  2. Strong proficiency in HTML, CSS, and JavaScript.
  3. Experience with responsive design and cross-browser testing.
  4. Understanding of RESTful APIs and version control systems (e.g., Git).

Preferable

  • Experience with React, Vue.js, or Angular frameworks.
  • Familiarity with CSS preprocessors (SASS/LESS) and front-end build tools (Webpack, Vite).
  • Knowledge of web accessibility (WCAG) and performance optimization best practices.
